# গল্পের.পাতায় অবদান

ধন্যবাদ গল্পের.পাতায় অবদান রাখতে চাওয়ার জন্য! 🎉

এই নির্দেশিকা আপনাকে নতুন গল্প জমা দিতে সাহায্য করবে।

---

## 📝 গল্প জমা দেওয়ার নিয়ম

### গল্প নির্বাচন

- ✅ গল্পটি **কপিরাইটমুক্ত** হতে হবে (লেখকের মৃত্যুর ৬০+ বছর পর)
- ✅ অথবা আপনি **লেখকের অনুমতি** নিয়ে থাকতে হবে
- ✅ অথবা গল্পটি আপনার **নিজের লেখা** হতে হবে
- ❌ কপিরাইটযুক্ত বাণিজ্যিক প্রকাশনা গ্রহণযোগ্য নয়

### ফাইল ফরম্যাট

প্রতিটি গল্প একটি `.md` (Markdown) ফাইল। ফাইলের শুরুতে YAML **frontmatter** থাকতে হবে:

```yaml
---
শিরোনাম: "গল্পের শিরোনাম"          # বাধ্যতামূলক
লেখক: "লেখকের পূর্ণ নাম"           # বাধ্যতামূলক
বিষয়শ্রেণী: "ছোটগল্প"               # বাধ্যতামূলক
ট্যাগ:                                # ঐচ্ছিক
  - "ক্লাসিক"
  - "বিষয়-ট্যাগ"
সংক্ষেপ: "গল্পের ২-৩ বাক্যে বিবরণ"  # প্রস্তাবিত
কপিরাইট_মুক্ত: true                  # বাধ্যতামূলক
প্রকাশকাল: "১৮৯২"                   # ঐচ্ছিক
উৎস: "মূল গ্রন্থের নাম"              # ঐচ্ছিক
এআই_নির্মিত: false                    # বাধ্যতামূলক (নিচে দেখুন)
---

গল্পের মূল লেখা এখানে শুরু হবে...
```

### 🤖 AI নির্মিত গল্পের নিয়ম

AI (যেমন ChatGPT, Gemini, Claude ইত্যাদি) দ্বারা তৈরি গল্প জমা দেওয়া যায়, তবে:

- ❗ **বাধ্যতামূলকভাবে** frontmatter-এ `এআই_নির্মিত: true` থাকতে হবে
- ❗ AI নির্মিত গল্প লুকিয়ে রাখা অগ্রহণযোগ্য
- ✔️ AI-তৈরি গল্প সাইটে 🤖 **AI** লেবেলসহ প্রদর্শিত হবে
- ✔️ AI দিয়ে লেখার পরও ভালো মানের গল্প প্রত্যাশিত

```yaml
# AI নির্মিত গল্পের উদাহরণ:
---
শিরোনাম: "স্বপ্নের নদী"
লেখক: "আপনার নাম"
বিষয়শ্রেণী: "ছোটগল্প"
এআই_নির্মিত: true
# ... অন্যান্য তথ্য
---
```

### বিষয়শ্রেণী

নিম্নলিখিত বিষয়শ্রেণী ব্যবহার করুন:

| বিষয়শ্রেণী | কখন ব্যবহার করবেন |
|------------|-------------------|
| ছোটগল্প | সাধারণ ছোটগল্প |
| উপন্যাস | দীর্ঘ গল্প / উপন্যাস |
| কবিতা | কবিতা ও পদ্য |
| প্রবন্ধ | প্রবন্ধ ও নিবন্ধ |
| নাটক | নাটক ও নাটিকা |
| রূপকথা | রূপকথা ও লোককথা |
| ভৌতিক | ভৌতিক ও রহস্য গল্প |
| হাস্যরস | হাস্যরসাত্মক গল্প |
| শিশু | শিশুতোষ গল্প |
| ঐতিহাসিক | ঐতিহাসিক ফিকশন |
| বৈজ্ঞানিক কল্পকাহিনী | সায়েন্স ফিকশন |
| অনুবাদ | অনুবাদিত সাহিত্য |
| স্মৃতিকথা | আত্মকথন ও স্মৃতিকথা |

```
golpo/
├── রবীন্দ্রনাথ-ঠাকুর/
│   ├── কাবুলিওয়ালা.md
│   └── পোস্টমাস্টার.md
├── শরৎচন্দ্র-চট্টোপাধ্যায়/
```

- ফাইল নাম = গল্পের শিরোনাম

## 🔧 কোড অবদান

কোড পরিবর্তনের জন্য:

1. Repository fork করুন
2. নতুন branch তৈরি করুন (`git checkout -b feature/your-feature`)
3. পরিবর্তন করুন
4. নিশ্চিত করুন `bun run build` সফলভাবে চলে
5. Pull Request পাঠান

### স্থানীয় ডেভেলপমেন্ট

```bash
bun install
bun run dev
```
